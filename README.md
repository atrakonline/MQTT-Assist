## Features

- Connect to a MQTT broker
- Subscribe to topics
- Real-time message viewer
- LED connection indicators
- Message length display
- Trial system with activation

## How to Use

1. Enter broker URL
2. Enter port and topic
3. Tap Connect
4. View incoming messages

For activation, contact via email inside the app.
