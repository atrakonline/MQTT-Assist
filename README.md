# MQTT-Assist

## Download

Download the latest version here:
https://github.com/atrakonline/mqtt-Assist/blob/main/MQTT_Assist_v1.0.apk

⚠️ Enable "Install from unknown sources" on your Android device.

Permissions: 
android.permission.WAKE_LOCK ,
android.permission.INTERNET ,
android.permission.ACCESS_NETWORK_STATE

Just an .apk file to install on Android 10+
The app. shows messages that sent to a TOPIC . The Log view (TextBox) can show 5000 chars (bytes) and cut the messages for the last 3000 chars. 
The decoding is UTF8 .
The App. has 15 days trial period . After Trial Period it shows 50 messages only and After this event just shows 3 last received message . Also showing messages not in real time and it's after 2 second

## Features

- Connect to MQTT brokers
- Subscribe to topics
- Real-time message viewer
- LED connection indicators
- Message length display
- Trial system with activation

## How to Use

1. Enter broker URL
2. Enter port and topic
3. Tap Connect
4. View incoming messages

For activation, contact via email inside the app.
